# Dataset

This project uses the Breast Cancer Wisconsin (Diagnostic) Dataset.

🔗 Dataset Link:
https://archive.ics.uci.edu/ml/datasets/Breast+Cancer+Wisconsin+(Diagnostic)

## Description
- Features: 30 numeric features computed from digitized images
- Target:
  - M = Malignant (1)
  - B = Benign (0)

## Note
The dataset is not uploaded in this repository due to size and licensing.
Please download it from the above link and place it in the `data/` folder.
